# Visual Question Answering App

A multimodal AI application that answers questions about images using Vision-and-Language Transformer (ViLT).

## How to Use

1. Upload an image
2. Ask a question about it (e.g., "What color is this?", "How many people are there?")
3. Get an AI-generated answer!

## Model

This app uses `dandelin/vilt-b32-finetuned-vqa`, a state-of-the-art Vision-and-Language Transformer.

## Examples

Try questions like:
- "What is in the image?"
- "What color is the object?"
- "How many people are visible?"
- "Is this indoors or outdoors?"
- "What is the person doing?"
